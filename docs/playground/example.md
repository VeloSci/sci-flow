## Features

- **Full Control Panel**: Manage nodes, edges, and themes.
- **Persistence**: Save and load your flow state as JSON.
- **MiniMap**: Real-time overview of your entire graph.
- **Advanced Nodes**: Includes Math, Generator, Processor, and Viewer nodes.
