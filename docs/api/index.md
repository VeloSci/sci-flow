# API Reference

Detailed documentation of all the classes, interfaces, and hooks available in `sci-flow`.

## Packages

- [**Core API**](/api/core): Agnostic logic, nodes, edges, and state management.
- [**React API**](/api/react): React components, hooks, and lifecycle management.

## Key Concepts

- **StateManager**: The source of truth for your graph.
- **NodeDefinition**: How to register and configure specific node types.
- **Port**: Input and output connection points.
- **SciFlowOptions**: Configuration for the main engine.
