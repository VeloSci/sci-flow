# SciFlow Playground

Experience the full power of SciFlow in this interactive playground. This demo includes custom React nodes, smart routing, state management, and more.

<SciFlowPlayground />

## Features in this Demo

- **Vibrant Custom Nodes**: Built entirely with React and mounted via Portals.
- **Smart Routing**: High-performance A* pathfinding for edges.
- **State Management**: Real-time synchronization between the agnostic core and React UI.
- **Theming**: Automatic adaptation to VitePress light/dark modes.
- **Persistence**: Save and load the entire flow state as JSON.

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| <kbd>Del</kbd> | Delete selected nodes/edges |
| <kbd>Shift + Drag</kbd> | Area selection |
| <kbd>Space + Drag</kbd> | Pan viewport |
| <kbd>Ctrl + Z</kbd> | Undo |
| <kbd>Ctrl + Y</kbd> | Redo |
